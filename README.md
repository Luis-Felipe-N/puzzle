# Quebra-Cabeça (Puzzle Game)

Um jogo de quebra-cabeça interativo desenvolvido com React, TypeScript e Vite. O projeto divide uma imagem em peças e as embaralha para que o jogador possa resolver.

## Funcionalidades

- **Geração de Peças**: Divide a imagem automaticamente em um grid baseado nas dimensões.
- **Embaralhamento**: Utiliza o algoritmo Fisher-Yates para embaralhar as posições das peças.

## Tecnologias Utilizadas

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## Como rodar o projeto

1. Instale as dependências:

```bash
npm install
```
